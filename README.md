# Run the project

```bash
./start-project.sh
```

# Stop running

```bash
./stop-project.sh
```

# API

| Method | Path                |
| ------ | ------------------- |
| GET    | /api/user/{id}      |
| POST   | /api/todo/{user_id} |
| PUT    | /api/todo/{user_id} |
| DELETE | /api/todo/{user_id} |

# Project description

- `Front-end`: React + MobX + Typescript
- `Back-end`: Go + Redis + MongoDB + Gin
